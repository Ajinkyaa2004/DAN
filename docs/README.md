# Documentation Organization

This folder contains all project documentation organized by category.

## Folder Structure

### 📦 deployment/
Contains all deployment-related documentation:
- `DEPLOYMENT_CHECKLIST.md` - Pre-deployment checklist
- `DEPLOYMENT_GUIDE.md` - Main deployment guide
- `DEPLOYMENT_URLS.md` - Production URLs and endpoints
- `HOW_TO_RUN.md` - Instructions for running the application
- `QUICK_START_DEPLOYMENT.md` - Quick start guide
- `README_DEPLOYMENT.md` - Deployment README
- `VERCEL_DEPLOYMENT_QUICK_START.md` - Vercel-specific deployment guide
- `YOUR_DEPLOYMENT_GUIDE.md` - Personalized deployment guide

### 🔧 development/
Contains development and integration documentation:
- `UNIFIED_DASHBOARD_SETUP.md` - Dashboard setup instructions
- `UNIFIED_INTEGRATION_GUIDE.md` - Integration guide for unified system

### 🐛 debug/
Contains debugging and troubleshooting documentation:
- `BRANCH_NAME_NORMALIZATION.md` - Branch name normalization issues
- `COMBINED_CSV_FIX.md` - CSV file processing fixes
