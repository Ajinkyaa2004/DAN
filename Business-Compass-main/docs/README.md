# Business Compass Documentation

This folder contains documentation specific to the Business Compass application.

## Folder Structure

### 🔧 development/
Development and implementation documentation:
- `DATA_PIPELINE_AUDIT.md` - Audit of the data processing pipeline
- `HARDCODED_VALUES_AUDIT.md` - Review of hardcoded values in the codebase
- `HOW_TO_UPLOAD_COMBINED_CSV.md` - Guide for uploading combined CSV files
- `IMPLEMENTATION_COMPLETE.md` - Notes on completed implementations

### 🐛 debug/
Debugging and troubleshooting guides:
- `ISSUE_DIAGNOSIS.md` - Diagnosis of known issues
- `QUICK_DEBUG_REFERENCE.md` - Quick reference for common debugging scenarios
- `REVENUE_DEBUG_GUIDE.md` - Guide for debugging revenue calculation issues
- `ROOT_CAUSE_FOUND.md` - Root cause analysis of past issues

## Main Application Files

- `README.md` - Main application README
- `package.json` - Project dependencies and scripts
- `next.config.ts` - Next.js configuration
- `tsconfig.json` - TypeScript configuration
- `vercel.json` - Vercel deployment configuration
